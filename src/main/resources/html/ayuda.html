Guía de Usuario y Documentación

Introducción

Esta aplicación permite gestionar alumnos, libros, préstamos y devoluciones de manera eficiente y amigable. A continuación se detallan los pasos para la instalación y las funcionalidades disponibles.

Instalación

Requisitos Previos

  • Java 8 o superior.
  • Base de datos MySQL o SQLite.
  • Herramientas de desarrollo como IntelliJ IDEA o Visual Studio.

Pasos para la instalación:

  1. Clonar el repositorio de GitHub.
  2. Configurar la base de datos con los scripts proporcionados.
  3. Importar el proyecto en tu entorno de desarrollo.
  4. Compilar y ejecutar el proyecto.

Funcionalidades

Gestión de Alumnos

Permite dar de alta, modificar y consultar la información de los alumnos registrados en el sistema.

Gestión de Libros

Los libros pueden ser dados de alta, modificados, y dados de baja. En caso de baja, el libro no se eliminará de la base de datos, sino que se actualizará su estado.

Gestión de Préstamos

Los libros se pueden prestar y consultar, siempre que no estén ya prestados. Cuando un libro se presta, se guarda el estado del préstamo y se genera un informe en PDF.

Gestión de Devoluciones

Permite registrar las devoluciones de libros, modificar su estado y moverlos al historial de préstamos.

Informes Generados

Informe 1: Préstamo de Libro

Cuando un libro se presta, se genera un informe en PDF que contiene los datos del préstamo, incluyendo el alumno, el libro, la fecha de préstamo y la fecha límite de devolución.

Informe 2: Listado de Libros

Este informe muestra un listado de todos los libros, con un subinforme que muestra los préstamos que ha tenido cada libro.

Informe 3: Gráficos de Préstamos

Genera varios gráficos que muestran la cantidad de veces que un libro ha sido prestado, entre otros datos estadísticos.

Informe 4: Informe de Alumnos

Este informe muestra información sobre los alumnos, como el número de libros que han tenido en préstamo y el tiempo total de préstamo.
